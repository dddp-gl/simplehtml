# SimpleHtml ## What is SimpleHTML ? Using SimpleHTML (an HTML document with additional functions), you can create documents conforming to the following formats. * Markup language (Markdown). * CSV (utf8) data output from Excel. * Diagram as code(Mermaid,PlantUML). * Graphs, data visualization, 3D animation using JavaScript (Chart.js, D3.js, Three.js). * Displaying formulas (MahtJax). * Diagram by GUI(Draw.io).
